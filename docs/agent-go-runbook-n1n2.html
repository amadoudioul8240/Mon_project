Runbook N1/N2 - Agent Go IT Monitoring

Version: 1.0.0 | Date: 02/04/2026

1. But

Ce runbook sert a exploiter, verifier et depanner rapidement l agent Go sur postes Windows.

2. Verification rapide (N1)

  1. Service present et demarre:
Get-Service ITMonitoringGoAgent
  1. Logs agent:
Get-Content C:\ProgramData\ITMonitoringAgent\agent.log -Tail 80
  1. Flux backend (UI):
Dual-Run Monitoring
- Etat machine: OK / Partiel / Alerte
- Dernier PS1 vs Dernier Go
Etat OK: service running + logs recents + donnees visibles sous 5 min.

3. Incidents standard et actions (N1)

SymptomeCause probableAction N1
Service absentInstallation non executeeRelancer install_service.ps1 en admin
Service stoppedCrash, binaire bloque, droitsStart-Service + verifier Event Viewer
Pas de remonteeBackend URL/proxy/firewallVerifier config.json + connectivite reseau
Partiel/Alerte dual-runUn seul agent actif ou staleVerifier timestamps, logs et statut agent cible

4. Diagnostics N2

4.1 Test endpoint backend

Test-NetConnection srv-it -Port 8000
Invoke-WebRequest http://srv-it:8000/docs -UseBasicParsing

4.2 Verifier config runtime

Get-Content C:\ProgramData\ITMonitoringAgent\config.json

4.3 Reinstallation propre

Set-Location C:\Temp\agent-go\deploy
.\uninstall_service.ps1
.\install_service.ps1

4.4 Verification dual-run backend

GET /agents/dual-run/health?active_minutes=60
GET /agents/dual-run/compare/{serial_number}

5. Escalade N3/SecOps

Escalader si: blocage AppLocker/EDR, signature invalide, conflit policy GPO, ou absence remontee globale.
  • Joindre logs agent (50-200 lignes)
  • Joindre event logs Service Control Manager
  • Joindre statut dual-run de la machine

6. Rollback d urgence

Si incident majeur, desactiver temporairement l agent Go et revenir au PS1.
.\uninstall_service.ps1
# puis relancer le flux PS1 selon procedure interne

7. Commandes utiles

sc.exe query ITMonitoringGoAgent
Restart-Service ITMonitoringGoAgent
Get-Content C:\ProgramData\ITMonitoringAgent\gpo-startup.log -Tail 50
Get-Content C:\ProgramData\ITMonitoringAgent\agent.log -Tail 100
