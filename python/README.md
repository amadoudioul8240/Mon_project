# Scripts Python

Ce dossier centralise les scripts Python autonomes du projet.

## Sous-dossiers

- `agent-linux/` : agent Linux Python.
- `agent-windows/` : variante Python de l'agent Windows legacy.
- `docs/` : scripts Python de generation documentaire.

## Exemples

```bash
python3 python/agent-linux/agent_linux_collect.py --verbose
```

```powershell
Set-Location python\agent-windows
python .\agent_collecte.py
```
